`0xFA6A9DdfD6006De9ea722CDB3c23ce34788c330B`

## Built By
[@cryptofunny724](https://x.com/cryptofunny724)

## Overview
Core contract responsible for tracking and managing user reputation scores across the ecosystem.

## Features
- `record_action(value)` — Increases or decreases reputation
- `my_score()` — View current reputation score
- Integrated with Interaction Hub and Task Marketplace

## Status
Stable and fully functional on GenLayer Bradbury Testnet.
